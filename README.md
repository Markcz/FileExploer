# FileExploer
# 1.主要使用了RecyclerView
# 2.实现的文件夹与文件的创建
# 3.通过文件类型的不同，创建了不同的Intent,实现Activity之间的通讯
# 4.实现了文件及目录的删除，移动以及复制
# 5.简单实现了Activity之间跳转的动画
